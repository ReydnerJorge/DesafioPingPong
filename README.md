# DesafioPingPong

### Objetivo Geral
 Criar um programa na linguagem Go que trabalhe com dois pacotes e concorrência.

### Desafio
<p>Escreva um código em GO utilizando todo o conhecimento adquirido até o momento!
E nesse código você precisará, baseado em nossas aulas de concorrência, utilizar canais e goroutines
para que o seu programa exiba, em alternância, as palavras ping e pong.</p>
